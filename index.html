Elchi Helm Repository

Official Helm charts for the Elchi proxy management platform

Quick Start

Add Repository

helm repo add elchi https://charts.elchi.io
helm repo update

Install Charts

# Complete platform
helm install my-stack elchi/elchi-stack --set global.mainAddress="your-domain.com"

# Discovery service only
helm install my-discovery elchi/elchi-discovery
