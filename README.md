## üöÄ Quick Start

### Add Repository
```bash
helm repo add elchi https://charts.elchi.io
helm repo update
```

### Install Charts
```bash
# Complete platform
helm install my-stack elchi/elchi-stack \
  --set global.mainAddress="your-domain.com"

# Discovery service only
helm install my-discovery elchi/elchi-discovery
```
